Was ist Web-to-Print?

Jede Webseite lässt sich ausdrucken!

Öffne ein beliebiges Dokument im Browser und wähle Datei → Drucken → Vorschau oder STRG + P, um zu sehen, wie es auf Papier aussehen würde. Wenn du als Ausgabe PDF auswählst, kannst du es dir als Datei ausgeben lassen.

Der Inhalt ist gleich, aber das Medium ein anderes! Anders als auf dem Bildschirm müssen wir dessen physische Eigenschaften Mediums berücksichtigen, wie: Papiergröße, Ränder, Seitenumbrüche und Druckfarben. Mit CSS haben wir die Möglichkeit, genau das zu tun!

Die Print-Funktion kann auch genutzt werden, um ein PDF zu generieren, dass vielleicht gar nicht für Print gedacht wird.

Code kann als Open-Source Gestaltungstool genutzt werden!

Media Queries

Media Queries erlauben es, CSS nur für bestimmte Medien anzuwenden. Für Druck ist das print-Medium wichtig:


@media print {
    body {
        font-size: 12pt;
        color: black;
        background: white; /* Hintergründe vermeiden */
    }

    nav, .no-print {
        display: none; /* Navigation und nicht-druckbare Elemente ausblenden */
    }
}

Man kann im Browser die Druckansicht simulieren, um ein Gefühl dafür zu bekommen, wie das Ergebnis aussieht. In Firefox geht das übr den Button Toggle print media simulation for the page drücken, um die Druckansicht zu simulieren.

  • Farben für besseren Druck vereinfachen
  • Navigation oder Menüs ausblenden
  • Hintergrundbilder entfernen
  • Seitenränder definieren

@page

@page ist eine CSS-Regel, die verwendet wird, um verschiedene Aspekte von Druckseiten zu ändern. Sie zielt auf die Dimensionen, die Ausrichtung und die Ränder der Seite ab und modifiziert diese. Die @page At-Regel kannverwendet werden, um alle Seiten in einem Ausdruck oder einen Teil mittels ihrer verschiedenen Pseudoklassen zu zielen.


/* Spricht alle Seiten an */
@page {
    margin-top: 4cm;
}

/* Spricht alle geraden Seiten an */
    @page :left {
    margin-top: 4cm;
}

/* Spricht alle ungeraden Seiten an */
@page :right {
    margin-top: 4cm;
}

/* Spricht alle Selektoren mit `page: wide;` an */
@page wide {
    margin-top: 1cm;
}

Druckformate

Mit CSS kann direkt das Format des Papiers definiert werden. Das funktioniert über @page

Format Abmessung Beschreibung
A4 210mm x 297mm Standardbrief und Dokumente
A3 297mm x 420mm Poster oder größere Layouts

Formate werden mit @page definiert.


@page {
    size: A4; /* oder A3, Letter, landscape */
    margin: 2cm; /* Seitenränder */
}

@page :landscape {
    size: A4 landscape;
    margin: 1in;
}

Bei den Margins ist es wichtig zu bedenken, dass jeder Drucker eigene Seitenränder erzeugt.

Seitenumbrüche

break-after

Seitenumbrüche verhindern ungewollte Trennungen von Absätzen, Überschriften oder Tabellen.


/* Seitenumbruch vor heder h2 Überschrift */
h2 {
    page-break-before: always; 
}

/* Seitenumbruch nach jedem Absatz */
p {
    page-break-after: always;
}

break-inside

Die Eigenschaft break-inside steuert, ob ein Element innerhalb einer Seite oder Spalte geteilt werden darf:

Wert Beschreibung
auto Ein Umbruch ist erlaubt, aber nicht erzwungen.
avoid Vermeidet Umbrüche innerhalb des Elements (Seite, Spalte oder Region).
avoid-page Vermeidet einen Seitenumbruch.
avoid-column Vermeidet einen Spaltenumbruch. 

p {
    break-inside: avoid;
}

Kopf- und Fußzeilen / Pagination

Mit CSS kann man Kopf- und Fußzeilen definieren und Seitenzahlen hinzufügen:


@page {
    margin: 2cm;

    @top-left {
        content: "Dokumenttitel";
    }

    /* Seitenzahl */
    @bottom-right {
        content: counter(page); 
    }
}
